# Fitness Tracker - Samsung Health Clone

A full-stack fitness tracking application built with React, Node.js, and Supabase.

## Features

- User authentication with email/password and social providers
- Activity tracking (workouts, steps, sleep, water intake)
- Goal setting and progress tracking
- Interactive charts and analytics
- Profile management with BMI calculator
- Admin panel for user management
- Dark/light mode support
- Responsive design

## Tech Stack

- Frontend: React + Vite + TailwindCSS + Zustand + Chart.js
- Backend: Node.js + Express.js
- Database & Auth: Supabase (PostgreSQL)
- Deployment: Vercel/Netlify + Supabase Cloud

## Setup Instructions

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Supabase account

### 1. Clone the repository

```bash
git clone <repository-url>
cd fitness-tracker
```
